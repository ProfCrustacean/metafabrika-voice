# Metafabrika Voice STT Service

Standalone browser-facing speech-to-text API for Russian (`ru-RU`) transcription.

## Start Here

- API + runtime setup: [`docs/stt-service.md`](./docs/stt-service.md)
- Repo navigation map: [`docs/REPO_MAP.md`](./docs/REPO_MAP.md)
- Agent workflow rules: [`AGENTS.md`](./AGENTS.md)

## Quick Commands

1. `npm install`
2. `cp .env.example .env`
3. Ensure `ffmpeg`/`ffprobe` are installed
4. `npm run check:all`
5. `npm run dev`
