# Integration Package: Metafabrika Voice STT (Stage 1)

Этот пакет предназначен для разработчика основного сервиса, который интегрирует голосовой ввод в существующий текстовый пайплайн.

## Состав

1. `01-architecture-and-flow.md` — архитектура, ключевые решения, рабочий флоу, ограничения.
2. `02-api-endpoints.md` — практический API-справочник по эндпойнтам с примерами запросов и ответов.

## Что важно помнить

- Это отдельный STT-сервис, который только превращает голос в текст.
- Сервис синхронный: один запрос -> один итоговый текст.
- Фронт отправляет короткий аудиофайл после push-to-talk.
- В stage 1 ориентируемся на короткие клипы и ограниченный rollout.
